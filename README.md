# PINNs-in-Groundwater

This repository provides a hands-on introduction to the application of Physics-Informed Neural Networks (PINNs) for solving groundwater flow problems. Designed for students, and researchers in hydrogeology, water resources, and scientific machine learning, the material combines groundwater flow theory with modern deep learning techniques.

Through a series of progressively structured examples, users will learn how to:

* Build neural networks using PyTorch
* Use automatic differentiation to compute spatial and temporal derivatives
* Formulate physics-based loss functions
* Incorporate boundary and initial conditions into model training
* Solve one-dimensional and two-dimensional groundwater flow problems
* Visualize hydraulic head distributions and groundwater flow fields


The repository emphasizes practical implementation and reproducible workflows, with detailed explanations of model architecture, training procedures, and post-processing techniques. Example notebooks guide users from basic PINN concepts to complete groundwater flow models, providing a foundation for more advanced applications such as transient flow, contaminant transport, unsaturated flow, and coupled hydro-environmental systems.
